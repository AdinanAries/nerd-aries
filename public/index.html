About

Hey, I'm Mohammed Adinan; call me Nerd Aries. Somewhere In 2011, I started out in IT as a computer hardware and networking engineer, and also a database administrator. Three years later, perhaps of doing mostly database stuff and writing SQL code, I figured my passion is programming, so I decided to go full-time in the direction of software.

Over the last five years, I've been actively writing code and building applications and websites. I've also helped so many beginners who mostly were students from various bootcamps and colleges who needed help as they struggled with their course materials.

I have both theoretical and practical knowledge on designing and creating software. I'm able to desing, build, deploy and maintain websites and applications. As a self-made software engineer and architect, I design system software architecture, build systems fullstack-wise with modern frontend tech, and usually a java and/or node.js backend, powered by Microsoft SQL server or MySQL database.
